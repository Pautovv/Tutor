# Ch3romaniiLearn

В данном проекте будет разработана система для обучения математике, физике и информатике, направленная на подготовку EГЭ.

Цель данного проекта - научиться строить SFT, RAG пайплайны, оборачивать полученные решение в vLLM/ollama для быстрого инференса, а так же писать писать fastapi-эндпоинты для общения с LLM.
